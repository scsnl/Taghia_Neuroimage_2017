An implementation of Bayesian Switching Factor Analysis (BSFA)

If you found it useful in your work, please cite: 
Jalil Taghia, Srikanth Ryali, Tianwen Chen, Kaustubh Supekar, Weidong Cai, Vinod Menon, Bayesian Switching Factor Analysis for Estimating Time-varying Functional Connectivity in fMRI, NeuroImage, Available online 4 March 2017, ISSN 1053-8119, http://dx.doi.org/10.1016/j.neuroimage.2017.02.083.

The implementation is heavily built on M.J. Beal's implementation of Variational Bayesian Mixture of Factor Analysers. Please cite their work as well. 
http://www.cse.buffalo.edu/faculty/mbeal/software.html
Ghahramani, Z. and Beal, M.J. (2000)
Variational Inference for Bayesian Mixtures of Factor Analysers
In Advances in Neural Information Processing Systems 12:449-455, eds. S. A. Solla, T.K. Leen, K, MIT Press, 2000.
